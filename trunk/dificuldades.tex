\chapter{Dificuldades Enfrentadas}

Encontramos certa dificuldade na hora de modelar as estruturas de dados para representar o problema de forma eficiênte.\\
\noindent Resolvemos por utilizar a estrutura de dados \textit{List}, que faz parte da biblioteca padrão do C++, por ter uma implementação otimizada para casos gerais, e ser de fácil manipulação ( possui interface rica, com métodos iteradores e métodos para inserção e remoção de elementos )\\
Tivemos algumas dificuldades na utilização da estrutura List, o que demandou pesquisa, mas acarretou em um bom aprendizado sobre a \textit{STL} ( Standard Template Library ) e a linguagem C++.\\
\noindent Ainda na STL, tivemos alguns problemas na hora de utilizar a função \textit{min\_element}, que foi necessário no nosso método Prim::extraiMinimo. Para utiliza-la, tivemos de sobrecarregar os operadores de comparação da classe NodeEdge.\\
